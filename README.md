This project demonstrates key filesystem operations in Linux, including device mounting, permission handling, and system-level administration tasks. It serves as a practical reference for working with block devices, mount points, and filesystem management.

Features

* Mounting and managing block devices

* Inspecting filesystem metadata

* Working with Linux permission models

* Clean, modular command-line workflows

* Clear documentation for learning and reuse

Technologies Used

* Linux / Bash

* Filesystem utilities (mount, lsblk, fdisk, etc.)

* Markdown documentation
